# Report Maker [OUTLOOK UPDATE]

Скрипт для создания отчётов о дежурстве.

Он сохраняет report.txt в той же папке, где находится программа. А также отправляет письмо через Outlook. (Получатель/и указываются в файле .py)

## Как пользоваться

1. Запустите скрипт.
2. Введите данные для отчёта.
3. Чтобы закончить ввод, введите стоп.
4. После этого будет создан файл report.txt. А также отправлено письмо через Outlook

## Особенности

- Если поле с сутью обращения оставить пустым и нажать Enter, скрипт подставит Не было звонков.
- Отчёт формируется автоматически после завершения ввода.

## Настройка параметров программного доступа (Outlook)
Этот метод самый безопасный избегания системных сообщений при отправке письма через скрипт, но, к сожалению, не всегда работает в последних версиях Outlook.

  - В Outlook откройте Файл → Параметры.

  - Перейдите в Центр управления безопасностью → Параметры центра управления безопасностью....

  - Выберите раздел Программный доступ.

  - Установите переключатель в положение "Никогда не предупреждать о подозрительных действиях (не рекомендуется)" и нажмите ОК.

## Примечание

Файл report.txt создаётся рядом со скриптом или .exe.
